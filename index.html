Definition List

Paired tag :
Padded tags are those tags which are required to be close at the end. The padded tag is closed with forward slash (‘/’) that is placed at before the tag name this padded tags are also called as paired, dependent, non singular, logical, container tags.
Unpaired tag :
Un Padded tags are those tags which are not required to be close at the end. This UN padded tags are also called as UN paired, independent, singular, physical, empty tags.
